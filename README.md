# أرشيفي 🎬

موقعك الشخصي لتتبع المسلسلات والأفلام والأنمي والكرتون والمسرحيات والقصص والألعاب — بحث يومي تلقائي عن الأخبار، تصحيح المطابقة، مجموعات أفلام، اقتراحات أعمال مشابهة، وتشفير اختياري لبياناتك. كل ذلك مجاني بالكامل، بدون أي تسجيل أو مفتاح API.

## ⚠️ مهم: وين تُحفظ بياناتك
تقدّمك وتقييماتك وملاحظاتك تُحفظ في `data/userdata.json` **داخل مستودعك على GitHub** — تفتح موقعك من أي جهاز وتشوف نفس البيانات. بما إن المستودع **عام** (شرط GitHub Pages المجاني)، هذي البيانات تكون مرئية لزوار المستودع، إلا إذا فعّلت **التشفير** (راجع لتحت) — لا يوجد نظام حسابات حقيقي (سيرفر+تسجيل دخول) لأن الموقع ثابت بدون سيرفر؛ التشفير هو البديل الواقعي الآمن لهذا القيد.

## مصادر البيانات لكل قسم
| القسم | المصدر | ملاحظة |
|---|---|---|
| أنمي | **Jikan (MyAnimeList)** أولًا | بوستر وقصة وحالة دقيقة لأنه مخصص للأنمي، مع اقتراحات وأعمال مشتقة |
| مسلسلات / كرتون | TVmaze | |
| أفلام | iTunes Search | |
| مسرحيات / قصص / ألعاب | يدوي بالكامل | لا توجد قاعدة بيانات مجانية بدون تسجيل لها |

## 1) النشر (مرة واحدة)
1. مستودع GitHub جديد **عام**. 2) رفع كل الملفات بهيكلتها. 3) **Settings → Pages** → Deploy from a branch → `main` / root. 4) **Actions** → شغّل "تحديث يومي" يدويًا أول مرة.

## 2) فعّل الحفظ والإضافة المباشرة
من الموقع ⚙: اربط `username/repo`، وأنشئ [Fine-grained Token](https://github.com/settings/personal-access-tokens/new) (Contents + Actions: Read/write، على مستودعك فقط). بدونه: بياناتك تُحفظ بمتصفحك مؤقتًا فقط، والإضافة تكون عبر صفحة GitHub Actions يدويًا.

## 🔒 التشفير (اختياري)
من ⚙ → "تفعيل التشفير" → اكتب كلمة مرور (لا تُحفظ بأي مكان إلا بمتصفحك للجلسة الحالية). من هذي اللحظة، `userdata.json` يُكتب كنص مشفّر (AES-GCM) غير قابل للقراءة بدون كلمة المرور. عند فتح الموقع من جهاز/متصفح جديد، يطلب منك كلمة المرور (زر "🔒 فتح بياناتي" بالشريط الجانبي). **لا تنسَ كلمة المرور — لا توجد طريقة لاستعادتها.** هذا تشفير حقيقي يحمي من المشاهدة العرضية، لكن كود فك التشفير نفسه علني (لازمة الموقع الثابت)، فهو غير مناسب لحماية بيانات حساسة فعليًا — كافٍ لمشروع شخصي.

## تصحيح البيانات الخاطئة
إذا جاب الموقع عمل غلط أو بوستر غير مناسب: افتح العمل → "نظرة عامة" → اكتب الاسم الصحيح بخانة "تصحيح المطابقة" → بحث → اختر النتيجة الصحيحة من القائمة (تظهر بوستراتها). يُثبَّت هذا الاختيار بالتحديثات القادمة فلا يرجع يجيب بيانات غلط. للأنمي، البحث يستخدم MyAnimeList تلقائيًا (بوسترات أنمي حقيقية لا بوسترات مسلسلات عامة).

## مجموعات الأفلام (سلاسل/فرنشايز)
داخل قسم "أفلام"، افتح أي فيلم → "نظرة عامة" → اكتب اسم مجموعة (مثل DC أو Marvel) واحفظ. بعدها يظهر شريط فلترة بأعلى قسم الأفلام بكل مجموعاتك، يبين كم شاهدت من كل مجموعة وكم باقي. لإخفاء فيلم قديم من "الباقي" بدون حذفه، علّمه "سحبت عليه" بزر ↩ على بطاقته.
> ملاحظة: اكتشاف كل أفلام سلسلة معيّنة تلقائيًا (مثل "كل أفلام DC حتى لو ما أضفتها") يحتاج قاعدة بيانات أفلام متقدمة (TMDB) وهي محظورة حاليًا على شبكتك. أضف أفلام المجموعة بنفسك عبر البحث، والموقع يتابع لك التقدّم تلقائيًا بعد ذلك.

## الاقتراحات والأعمال المشتقة
لكل أنمي، تبويب "اقتراحات" بتفاصيل العمل يعرض: الأعمال المشتقة منه (تتمة/جزء سابق/قصة جانبية...) وأعمال مشابهة يقترحها مستخدمو MyAnimeList — بضغطة "➕ أضف" تضاف لأرشيفك تلقائيًا ببياناتها الصحيحة. غير متاح تلقائيًا لباقي الأنواع (يحتاج قاعدة بيانات غير متوفرة مجانًا)، استخدم المجموعات اليدوية بالأفلام كبديل.

## تصحيح الحالة تلقائيًا
لو غيّرت حالة عمل يدويًا (مثلًا لـ"مستمر") والبيانات التلقائية كانت تقول نفس الشي وقتها، يبقى تعديلك فعّال. لكن لو تغيّرت الحقيقة فعليًا بعدين (مثلًا انتهى المسلسل واكتمل)، يتجاوز التحديث التلقائي تعديلك القديم ويعرض الحالة الصحيحة الجديدة — تعديلك اليدوي مؤقت لحين ما تتغيّر الحقيقة، مو دائم.

## حالة "غير مكتمل"
خيار بقائمة "حالة المتابعة"، لحالتين: (1) يدويًا لأعمال توقفت بالنص بدون نهاية واضحة (سيراف النهاية، هنتر اكس هنتر)، (2) تلقائيًا لو عمل "مكتمل" عندك صدر له جزء/حلقات جديدة.

## باقي الميزات
- **📰 الأخبار**: قسم منفصل يجمع كل الأخبار من كل أعمالك، 3 فئات (تجديد/إلغاء، مواعيد/تأجيل، عام).
- **شارة الحالة + عدد الأجزاء على البطاقة**: تشوفها بدون فتح العمل.
- **أزرار سريعة ✓/↩**: علّم "شاهدته"/"سحبت عليه" من البطاقة مباشرة.
- **نقل بين التصنيفات**: تفاصيل العمل → نظرة عامة → اختر تصنيف → نقل.
- **الإضافة/التحديث**: كل الأزرار بالموقع تُشغّل GitHub Action مباشرة عبر API (بدون فتح GitHub) عند تفعيل التوكن.

## حذف عمل
"حذف نهائي من المصدر" يفتح `titles.json` لحذف السطر، أو "إخفاء من أرشيفي" للإخفاء بمتصفحك فقط.

## النسخ الاحتياطي
"تصدير بياناتي" يحفظ تقدّمك (بدون التوكن أو كلمة التشفير لأمانك). "تصفير بياناتي" يمسح كل شي من المستودع والمتصفح.

## ملاحظات
- المستودع لازم يكون **عام**. لا تفتح `index.html` بنقر مزدوج مباشر — استخدم `npx serve .` أو انشره على Pages. التكلفة: صفر بالكامل.
